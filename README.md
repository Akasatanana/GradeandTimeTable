# 時間割＆GPA
時間割にGPAの自動計算を行う機能を追加したアプリです．
## 使い方
### 授業の登録
時間割ページで授業のコマをタップすると，授業の設定ができます．

項目を入力したら，メニューボタンから授業を登録しましょう．
